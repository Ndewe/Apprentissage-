#plateforme d'apprentissage pour lycéens 
Plateforme web complète permettant aux lycéens d'accéder à des cours, exercices et anciens sujets d'examen et leurs corrigés au format PDF. système de paiement avec OM et MTN Mobile Money pour d'autres corrigés 

## Fonctionnalités 

### pour utilisateurs (élèves)
- Inscription et connection sécurisé
- Consultation des documents
- Recherche et filtrages par niveaux, matières et types
- Ajout des documents au favoris
- Panier d'achat
- Paiement par OM ou MTN Mobile money
- Téléchargement du document acheter
- Espace personnel avec historique des achats
- Notification en temps réel

### Pour les administrateurs
- Tableau de bord avec statistiques
- Gestion des documents ( ajout, modification et suppression)
- Import des documents
- Gestion des utilisateurs
- Gestion des commandes
- Vérification des transactions Mobile
- Paramètres du site

## Technologies utilisées 
- Serveur local: Xampp
- Langage de programmation : HTML, php, JavaScript et CSS
- Base de données : My SQL
